 O CurryGerador é um site criado para ajudar pessoas a montar seu currículo de forma simples, rápida e prática.  
A ideia é que qualquer pessoa consiga preencher seus dados — como informações pessoais, formação, cursos e objetivo profissional e gerar um currículo pronto para imprimir ou baixar em PDF.
---
Tecnologias usadas:
- **PHP** para processar os dados do formulário e montar o currículo.  
- **HTML e CSS** para a estrutura e o visual do site.  
- **JavaScript** para as interações e geração de PDF.  
- **jsPDF e html2canvas** para transformar o currículo em PDF.  
- **Bootstrap Icons** para os ícones da interface.
---
Como rodar o projeto (passo a passo):
1. Instale o [XAMPP](https://www.apachefriends.org/pt_br/index.html). 
2. Copie a pasta do projeto para dentro de:C:\xampp\htdocs\currygerador
3. Abra o painel do XAMPP e inicie o **Apache**.
4. No navegador, acesse:http://localhost/currygerador
5. Clique em **“Criar meu currículo”** e comece a preencher!
